# ProyectoFinal

🎯OBJETIVOS
El principal objetivo de este proyecto de programación  es  crear una app  simuladora  con interfaz gráfica en java en donde  se realice una importación completa de datos desde leer los archivos hasta que se almacenen en la base de datos.

📌REQUISITOS.

1.La base de datos debe de contener minimamente dos tablas y 10 registros en total ademas de que los atributos deben de ser distintos es decir no todos los atributos deben de ser de tipo VARCHAR(), las tablas deben de contener clave primaria y alguna conexion entre ellas .
2.La applicación deben de permitir realizar búsquedas mínimas de algún registro que exiga el usuario por ejemplo de algún producto comprado.
3.La applicación debe permitir al usuario eleminar algún registro deseado.
4.La applicación debe permitir  buscar productos por fechas.
5.La applicación debe tener incorporado una calculadora para realizar cualquier operación aritmética
6.La applicación deber de tener como  mínimo dos usuarios de acceso uno admin y otro cliente.
7.El programa debe de estar bien organizado con comentarios incluidos de aquellas partes más compleja en clases y paquetes para ello previamente se realizara un diagrama de clases.
8.La applicación debera estar formada por dos ventanas como mínimo y esten relacionadas entre sí.
9.El programa deberá permitir acceso a las diferentes interracciones con la base de datos por ejemplo si el usuario quisiera añadir otro usuario.
10.Al finalizar la applicación se deberá mostrar un mensaje de finalización de programa.
